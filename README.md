# House Rent Prediction System

This is a Machine Learning based web application that predicts house rent.

## Features
- Predicts monthly rent
- Uses Linear Regression
- Dynamic city and locality selection
- Area-based validation
- Built with Streamlit

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run Locally

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## Files
- app.py : Main application
- linear_model.pkl : Trained ML model
- cities_magicbricks_rental_prices.csv : Dataset
- requirements.txt : Dependencies
